# 영화 · TV · 인물 정보 제공 사이트

URL: [https://tvie-one.vercel.app/](https://tvie-one.vercel.app/)  
Git: [https://github.com/ggj102/Tvie](https://github.com/ggj102/Tvie)

---

## 기간

- 2023.11.2 ~ (진행 중)
- 2023.11.15 첫 배포

## 프로젝트 소개

![tvie_image](/static/images/tvieImg.PNG)

- TMDB 사이트와 유사한 영화 및 TV 정보 플랫폼을 구현하였습니다. TMDB에서 제공하는 Open API를 기반으로 영화, TV, 인물과 관련된 다양한 정보를 제공하는 페이지를 개발하였습니다.

## 주요 기능

- 영화, TV, 인물 관련 정보 표시
- 로그인 기능(Auth0 사용)
- 각종 컨텐츠 및 인물에 대한 상세 정보 제공
- 검색과 다양한 필터 제공
- 최근 본 목록 제공
- 로그인 시 즐겨찾기 제공

## 실행 가이드

```
// 설치
npm install

// 개발 모드
npm run dev

// 빌드
npm run build

// 프로덕션 모드
npm run start
```

## 페이지 구성

- Home
- Contents (Movie/TV)
- Person
- Contents Detail
- Person Detail
- Search Results
- History
- Favorites
